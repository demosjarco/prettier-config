# `@demosjarco/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ npm i -D @demosjarco/prettier-config
```

**Edit `package.json`**:

```jsonc
{
	// ...
	"prettier": "@demosjarco/prettier-config"
}
```
